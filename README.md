# Gooey-Radial-Button-Menu

Gooey-Radial-Button-Menu ist ein CSS-Only-Menü.

## Installation

Benutze den package manager NPM um die notwendigen Abhängigkeiten.

```bash
npm install
```

## Benutzung

Entwicklung:

```bash
npm run dev
```
